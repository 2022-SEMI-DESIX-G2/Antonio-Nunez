JavaScript es un lenguaje de programación o de secuencias de comandos que permite implementar funciones complejas en paginas web, ademas de animaciones 2d, 3d, reproductores, mapas interactivos, actualizacion de contenido y demas. Es la tercera capa del pastel de programacion web asi como HTML y CSS.

Javascript y Java tienen en comun que son lenguajes de programación y solo su nombre que comienzan con Java.
